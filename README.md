# 🚀 Java Practice Programs

This repository contains a collection of **Java programs** focused on strengthening core programming logic and problem-solving skills. These programs cover basic to intermediate concepts commonly asked in coding interviews and academic exams.

---

## 📌 Topics Covered

- 🔢 Number-based problems  
- 🔁 Arrays and subarrays  
- 🔤 Strings and pattern matching  
- 📐 Mathematical logic  
- 🔄 Array rotations and transformations  
- 📊 Frequency and counting problems  

---

## 📂 Program List

### 🔢 Number-Based Programs
- `ARMSTRONG.java` → Check if a number is an Armstrong number  
- `PERFECT.java` → Check if a number is a Perfect number  
- `PRIME.java` → Check whether a number is Prime  
- `STRONGNO.java` → Check for Strong number  
- `REVERSEINTEGER.java` → Reverse an integer  

---

### 🔁 Array-Based Programs
- `LEADERSARRAY.java` → Find leaders in an array  
- `LEFTROTATEKSTEPS.java` → Rotate array by K steps  
- `MOVEALLZEROS.java` → Move all zeros to the end  
- `SUBARRAY.java` → Subarray-related operations  

---

### 🔤 String-Based Programs
- `LONGESTSUBSTRING.java` → Find longest substring  
- `STRINGCOMP.java` → String comparison logic  

---

### 📊 Mathematical & Utility Programs
- `GCDLCM.java` → Compute GCD and LCM  
- `FREQELEMENTS.java` → Frequency of elements in array  
- `DUPLICATE.java` → Find duplicate elements  

---

### 🎨 Pattern Programs
- `HOLLOWPYRAMID.java` → Print hollow pyramid pattern  

---
